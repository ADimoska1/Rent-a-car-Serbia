# Car Images

Place your car images in this folder with the following names:

- `skoda-fabia.jpg` - Image for Skoda Fabia
- `vw-golf.jpg` - Image for Volkswagen Golf  
- `fiat-punto.jpg` - Image for Fiat Punto

**Image Requirements:**
- Recommended size: 600x400 pixels or larger
- Format: JPG, PNG, or WebP
- File size: Keep under 500KB for fast loading

If images are not found, the website will automatically use placeholder images.
